# Site Evozzn

Site institucional e portfólio da agência Evozzn.

## Tecnologias

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Font: Inter

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador.

## Build

```bash
npm run build
```
